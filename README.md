# Manahil Khattak — Personal Portfolio

A responsive personal portfolio website built with pure HTML and CSS.

## Pages
- **Home** — Introduction, skills overview, and about section
- **Profile** — Education, experience, and achievements
- **Gallery** — Photo highlights
- **Goals** — Short and long-term career goals
- **Projects** — Showcased personal and academic projects
- **Skills** — Technical skills with progress bars and personal interests
- **Contact** — Contact form and social links

## Tech Stack
- HTML5
- CSS3 (Flexbox, Grid, Media Queries)
- No frameworks or libraries

## How to Run
1. Clone or download the repository
2. Open `index.html` in any browser
